可用事件类型列表

以下列表中的事件类型均可以在其“应用于”的对象上调用 on、off 或 fire 方法时使用，如：

window.fire('resize);
document.off('click.foo');
$element.on('mousedrag:relay(.draggable)', function(e) {...});

除了这里列出的 DOM 事件类型外，OurJS 还支持自定义 DOM 事件类型。
为自定义事件类型命名时，注意不要使用以下列表中已存在的事件类型，以免事件被意外的触发。

关于可以阻止默认行为的事件类型：

  • contextmenu：
    IE8 只要添加了该类事件的监听器，即自动取消默认行为。
  • keydown & keypress：
    大部分浏览器都禁止取消特殊按键（如 F5 和 Backspace 等）的默认行为。

关于 error 事件类型：

  • IE6 IE7 IE8 在 window 上添加的原生监听器会传入三个参数 message、url 和 lineNumber，而不是一个事件对象。鉴于极少监听此类型的事件，OurJS 未做修复。
    必须要监听时，可以使用内联事件监听器，这种监听器在目前所有浏览器下均与上述 IE6 IE7 IE8 的行为保持一致。

Opera 不支持的事件类型：

  • unload & beforeunload
